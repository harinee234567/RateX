# Currency Converter Pro

Currency Converter Pro is a modern Chrome extension for fast and intelligent currency conversion. It supports multiple conversion modes, over 150 currencies (including cryptocurrencies), and features a clean, dark-themed interface.

## Features
- Support for 150+ fiat and cryptocurrencies
- Automatic currency detection and page-wide conversion
- Selection-based conversion with tooltip display
- Manual conversion using a built-in calculator
- Batch conversion for multiple values
- Pinned favorite currencies for quick access
- Smart caching and automatic rate updates
- Privacy-focused with no tracking or data collection

## Conversion Modes
- **Auto Mode** – Automatically converts currencies on web pages
- **Selection Mode** – Converts selected text instantly
- **Manual Mode** – Calculator-based conversion with quick currency swapping

## Settings & Customization
- Default base currency
- Decimal precision control (0–6 places)
- Rate adjustment to account for exchange fees
- Manual and automatic exchange rate refresh
- Geo-based currency suggestions

## Exchange Rate APIs
- ExchangeRate-API (primary)
- Frankfurter API (fallback)
- ExchangeRate.host (fallback)

Rates are cached locally and refreshed hourly.

## Privacy
- No analytics or user tracking
- No personal data stored
- All processing handled locally

## Installation
1. Clone or download this repository
2. Open Chrome and navigate to `chrome://extensions`
3. Enable Developer Mode
4. Click **Load unpacked** and select the extension directory.
